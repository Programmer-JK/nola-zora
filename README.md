# 🎮 놀아조라 ARCADE

> 친구들과 즐기는 보드게임 아케이드 허브 — 설치 없이 브라우저에서 바로 플레이

레트로 아케이드 감성의 UI(CRT 화면 효과 · 네온 글로우 · 픽셀 폰트)로
8가지 보드게임을 온라인 또는 로컬로 즐길 수 있는 웹 앱입니다.

## 🚀 빠른 시작 — 친구 초대 방법

### 1단계 · 닉네임으로 입장
사이트에 접속하면 닉네임 입력창이 나타납니다.
원하는 닉네임(최대 12자)을 입력하고 **INSERT COIN** 버튼을 누르면 즉시 입장합니다.
회원가입, 로그인, 앱 설치 없이 바로 시작됩니다.

### 2단계 · 로비에서 게임 선택
로비에서 하고 싶은 게임 카드를 탭합니다.

### 3단계 · 방 만들기 & 초대

#### 온라인 플레이
```
방장이 "새 방 만들기" 클릭
  → 6자리 방 코드 자동 생성 (예: AB3X7K)
  → 코드 또는 QR을 친구에게 공유
  → 친구는 같은 게임 페이지에서 코드 입력 후 참가
  → 모든 인원이 모이면 방장이 "게임 시작"
```

> **QR 공유 팁**: 방 대기실에서 QR 코드를 화면에 띄우면
> 친구가 카메라로 스캔해 곧바로 입장할 수 있습니다.

#### 로컬 플레이 (한 기기에서 진행)
```
게임 선택 → "로컬" 탭 선택
  → 인원 수 설정 (게임마다 최대 인원 다름)
  → 게임 시작 → 기기를 차례로 넘겨가며 플레이
```

## 🎲 게임별 상세 소개

### 🕵️ 라이어 게임 `3–8인` `소셜 디덕션`
단어를 모르는 **라이어**를 찾아내는 눈치게임.

**게임 흐름**
1. 라운드 시작 시 카테고리와 단어가 공개됩니다.
2. 시민은 단어를 볼 수 있고, 라이어는 카테고리만 알 수 있습니다.
3. 모두가 순서대로 단어를 설명하는 힌트를 하나씩 말합니다.
4. 자유 토론 후 투표로 라이어를 지목합니다.
5. 지목된 라이어는 단어를 맞힐 기회! 맞히면 **라이어 역전승** 🎉

**점수**
- 라이어 탈출 성공 or 단어 맞히기 → 라이어 **+2점**
- 시민이 라이어를 잡고 단어도 틀림 → 시민 전원 **+1점**

**특별 모드**
- ⚡ **거짓 라이어**: 라이어도 본인이 라이어인 줄 모릅니다. 시민과 다른 단어를 받아요. 라이어가 없는 라운드도 있을 수 있어요!

---

### ⛵ 요트 `1–6인` `주사위 족보`
주사위 5개로 12가지 족보를 채워 최고 합계를 기록하는 게임.

**게임 흐름**
1. 내 차례가 되면 주사위 5개를 굴립니다 (한 턴 최대 3번).
2. 남기고 싶은 주사위를 탭해 **HOLD** — 고정된 주사위는 다음 굴림에서 제외.
3. 굴림이 끝나면 족보 12칸 중 하나를 선택해 기록합니다.
4. 12칸을 모두 채우면 합산 최고점 플레이어 승리!

**족보 구성**
| 상단 (6칸) | 하단 (6칸) |
|-----------|-----------|
| 1~6 각 눈의 합 | 초이스 · 포카드 · 풀하우스 |
| 상단 합계 63+ → **보너스 +35점** | 소스트레이트 · 대스트레이트 · **요트 50점** |

> 1인 솔로 모드도 지원 — 내 최고 기록에 도전!

---

### 🎲 라스베가스 `2–6인` `주사위 전략`
카지노에 주사위를 배치해 가장 많은 돈을 쓸어담는 게임.

**게임 흐름**
1. 각 플레이어는 여러 개의 주사위를 가지고 시작합니다.
2. 자기 차례에 주사위 전부를 굴려, 같은 숫자의 주사위를 해당 카지노에 배치합니다.
3. 라운드 종료 후 각 카지노에서 가장 많은 주사위를 가진 플레이어가 상금을 가져갑니다.
4. 4라운드 후 가장 많은 돈을 모은 플레이어 승리!

---

### 🃏 캐릭터 카드 `2인+` `파티 카드`
무작위 카드를 조합해 나만의 캐릭터를 만들어 설명하는 파티 게임.

**게임 흐름**
1. 각자 무작위 카드를 여러 장 받습니다.
2. 자신의 카드 조합으로 독특한 캐릭터를 만들어냅니다.
3. 다른 플레이어들 앞에서 창의적으로 설명합니다.
4. 가장 재미있는 캐릭터를 만든 사람이 승리!

---

### 🤔 나는 누구? `2–8인` `소셜 추리`
이마에 붙은 단어를 질문으로 추리하는 게임.

**게임 흐름**
1. 방에 입장 후 채팅으로 협의해 다른 플레이어의 이마에 붙일 단어를 선정합니다.
2. 모두의 단어가 확정되면 게임 시작 — 내 이마의 단어는 보이지 않습니다!
3. 채팅으로 다른 플레이어에게 **예/아니오** 질문을 하며 내 단어를 추리합니다.
4. 정답을 알 것 같으면 정답 칸에 입력해 제출합니다.

**오답 페널티**
| 오답 횟수 | 페널티 |
|---------|-------|
| 1회 | 없음 |
| 2–3회 | 10초 대기 |
| 4회 | 30초 대기 |
| 5회 | 1분 대기 |
| 6회+ | 2분 대기 |

> 팁: 단어 선정 단계에서 채팅에 단어를 직접 쓰지 마세요 — 해당 플레이어도 채팅을 볼 수 있습니다!

### 🔮 아브라카왓 `2–6인` `마법 블러핑`
숨겨진 타일을 추리하고 주문을 선언해 라이벌 마법사를 쓰러뜨리는 게임.

**게임 흐름**
1. 1~8번 타일 36장 중 각자 5장을 비밀리에 받습니다 (숫자 클수록 장수 많음).
2. 내 차례에 1~8 중 원하는 숫자를 **선언**합니다.
   - 그 타일을 실제로 갖고 있으면 **성공** → 타일 제거 & 주문 발동, 이하 숫자로 콤보 가능!
   - 없으면 **실패** → 체력 -1, 턴 즉시 종료.
3. 체력이 0이 되면 탈락. 생존자가 1명 이하거나 손패를 모두 비우면 라운드 종료.

**점수**
- 생존: **+1점** / 손패 모두 비우기: **+3점** / 천리안(4번) 사용: **+1점**
- 설정한 라운드(1~4) 종료 후 최다 점수자 승리!

**8가지 주문** — 전원 공격, 회복, 주사위, 인접 공격 등 각 숫자마다 강력한 마법 효과!

---

### 🎨 모던 아트 `2–5인` `경매 전략`
5명의 작가 작품을 경매로 사고팔아 최고의 자산가가 되는 게임.

**게임 흐름**
1. 내 차례에 손패에서 카드 1장을 골라 경매에 올립니다.
2. 경매 방식은 카드에 따라 다릅니다:
   - 📣 **공개 경매**: 시계방향으로 돌아가며 입찰. 최고가 낙찰.
   - 🏷️ **지정가 경매**: 판매자가 가격을 먼저 정하고 순서대로 수락/거절.
   - 🤫 **비밀 경매**: 전원 동시 비밀 입찰 후 공개. 최고가 낙찰.
   - 🔄 **한 바퀴 경매**: 각자 한 번만 입찰 기회. 판매자는 마지막에 이기거나 직접 구매.
   - 🎴 **더블 경매**: 같은 작가 카드 2장을 동시에 경매.
3. 한 작가의 카드가 5장 나오면 라운드 즉시 종료. 많이 팔린 TOP 3 작가 작품 가치 상승.
4. 총 4라운드 후 **현금 + 작품 가치** 합산 최다 보유자 승리!

---

### 🐉 구룡투 `2인` `1v1 전략 대결`
1~9 숫자 타일로 싸우는 2인 전략 게임. (Best of 3)

**게임 흐름**
1. 각 플레이어는 1~9 타일 9개를 받습니다 (홀수 = 흑, 짝수 = 백).
2. 공격자가 타일 1개를 뒷면으로 놓고, 수비자도 놓은 후 동시에 공개합니다.
3. 숫자가 더 큰 타일을 낸 사람이 라운드 승리.
   **⚡ 단, 1이 9를 이깁니다 — 용의 비밀무기!**
4. 9라운드 중 5라운드를 먼저 이기면 즉시 게임 종료.
5. 2게임을 먼저 이기는 사람이 최종 승자!

## 🚀 개발 시작하기

### 요구 사항
- Node.js 18+
- Firebase 프로젝트 (Anonymous Auth · Firestore · Realtime Database 활성화)

### 설치 및 실행

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속

### 환경 변수

프로젝트 루트에 `.env.local` 파일 생성:

```env
NEXT_PUBLIC_FIREBASE_API_KEY=...
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=...
NEXT_PUBLIC_FIREBASE_DATABASE_URL=...
NEXT_PUBLIC_FIREBASE_PROJECT_ID=...
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=...
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=...
NEXT_PUBLIC_FIREBASE_APP_ID=...
```

### 빌드

```bash
npm run build
npm run start
```

## 📁 프로젝트 구조

```
app/
├── page.tsx              # 로그인 (닉네임 입력)
├── lobby/                # 게임 선택 로비
├── liar/                 # 🕵️ 라이어 게임
│   ├── page.tsx          # 설정 (로컬/온라인, 모드)
│   ├── room/[code]/      # 대기실
│   ├── game/[code]/      # 게임 화면
│   ├── game/local/       # 로컬 게임
│   ├── rules/            # 규칙 설명
│   └── words/            # 단어 목록
├── yacht/                # ⛵ 요트
├── las-vegas/            # 🎲 라스베가스
├── card-game/            # 🃏 캐릭터 카드
├── who-am-i/             # 🤔 나는 누구?
├── abra/                 # 🔮 아브라카왓
├── modern-art/           # 🎨 모던 아트
└── guryongtu/            # 🐉 구룡투
```

각 게임 디렉토리는 `page.tsx`(설정) → `room/[code]/`(대기실) → `game/[code]/`(게임) → `rules/`(규칙) 흐름을 따릅니다.
